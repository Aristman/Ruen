# RuEn

Учебный финальный проект по курсу "Андроид-разработка" на портале GeekBrains

## Используемый стек технологий

1. Архитектура
  - Clean Code
  - MVVM
2. Data
  - Retrofit
  - Room
3. Base Structure
  - Single Activity
  - Fragments
  - KTX Library
  - Coroutines
  - Kotlin Flow
4. UI
  - Glide
5. DI
  - Hilt
